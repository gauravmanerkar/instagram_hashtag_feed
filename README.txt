INTRODUCTION
------------

This module pull data from Instagram based on hashtag.
 
 * For a full description of the module visit:
   https://www.drupal.org/project/instagram_hashtag_feed

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/instagram_hashtag_feed

REQUIREMENTS
------------

This module requires no modules outside of Drupal core.

INSTALLATION
------------
 
 * Install the Instagram Instagram Hashtag Feed as you would normally 
   install a contributed Drupal module. Visit 
   https://www.drupal.org/node/1897420 for further information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module.
    2. Navigate to Administration > Structure > Block Layout and place a block.
    3. Search for the "Instagram Hashtag Feed" block and assign it to a page
       and Save.
    4. Navigate to Administration > Configuration > System > Instagram Hashtag 
       Feed for more configurations.
